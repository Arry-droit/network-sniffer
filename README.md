# Network Sniffer

A Java-based network packet sniffer application.

## Description

This project provides network packet sniffing capabilities using Java.

## Features

- Packet capture
- Protocol analysis
- Packet filtering
- Basic statistics

## Requirements

- Java 11 or higher
- Maven

## Building

```bash
mvn clean install
```
